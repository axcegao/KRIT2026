Проект: "Кибербезопасность" — веб-сайт для конкурса «КРИТ-2026»
Номинация: «Веб-проект: сайт с нуля»
Автор: Шайхутдинов Урал
МАОУ Гимназия №3

Суть проекта:
Сайт содержит научно-популярную информацию о кибербезопасности, включая советы по созданию
надёжных паролей и распознаванию фишинга. Он направлен на повышение осведомлённости пользователей о важности защиты личной информации в интернете.

Задачи проекта:
Предоставить доступ к актуальной информации о киберугрозах и методах защиты.
Реализовать интерактивный элемент — проверку силы пароля в реальном времени, чтобы пользователи могли оценить свои пароли.
Создать визуально привлекательный интерфейс с использованием технологий веб-дизайна.
Обеспечить удобство навигации и доступность информации для широкой аудитории.

Реализован интерактивный элемент — проверка силы пароля в реальном времени.
Фон с частицами реализован через библиотеку particles.js.
Дизайн выполнен в технологичном стиле с hover-эффектами и анимациями.

Состав и назначение файлов:
index.html — главная страница сайта с основной информацией и интерактивными элементами.
style.css — файл стилей, содержащий все необходимые правила для оформления сайта.
script.js — файл JavaScript, реализующий логику проверки пароля и взаимодействие с библиотекой particles.js.
background.js - файл, отвечающий за настройку и управление фоновыми эффектами на сайте.
Взаимодействует с библиотекой particles.js, обеспечивая плавное отображение анимации частиц на фоне.
particles.min.js — библиотека для создания анимации частиц на фоне сайта.

Использованные технологии:
- HTML5
- CSS3 (Flexbox, градиенты, анимации)
- Vanilla JavaScript
- Библиотека particles.js (не является фреймворком)

Личный вклад:
- Вся разметка, стили и логика написаны с нуля.
- Подключение и настройка particles.js выполнены вручную.
- Интерактивный пароль-чекер реализован без внешних зависимостей.
